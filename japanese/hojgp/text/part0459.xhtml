【てこそ】

［V-てこそ］

（1）一人ひとりでやってこそ身みにつくのだから、むずかしくてもがんばってやりなさい。／Keep working on it, even if it’s hard, because it’s only by doing it yourself that you will learn.
（2）この木きは雨あめの少すくない地方ち ほうに植うえてこそ価値か ちがある。／It’s only worth planting this tree in areas with little rain.
（3）互たがいに助たすけ合あってこそ本当ほん とうの家族か ぞくといえるのではないだろうか。／Isn’t it only by helping each other that you can be called a real family?
（4）この問題もん だいは皆みなで話はなし合あってこそ意味い みがある。規則き そくだけ急いそいで決きめてしまうというやり方かたには反対はん たいだ。／This issue has meaning only if everyone discusses it together. I’m opposed to rushing to make decisions only about rules.
（5）今いまあなたがこうして普通ふ つうに暮くらせるのは、あの時ときのご両親りょう しんの援助えん じょがあってこそですよ。／The reason you can live normally now is only because of your parents’ support at that time.
In this expression, こそ is attached to the verb te-form for emphasis. A positive expression follows V-てこそ to express the meaning “only by doing ... does it become meaningful and render a good result.” When it expresses the reason for the positive result, as in example (5), こそ can be replaced by ばこそ.
（例）ご両親の援助があればこそです。／It is completely owing to your parents’ support.
