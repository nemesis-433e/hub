【んじゃ】

［N／Na 　（なん）じゃ］

［A／V　んじゃ］

（1）雨あめなんじゃしかたがない。あしたにしよう。／It’s no use if it’s raining. Let’s do it tomorrow.
（2）そんなに臆病おく びょうなんじゃ、どこにも行いけないよ。／If you’re so cowardly, you can’t go anywhere.
（3）こんなに暑あついんじゃ、きょうの遠足えん そくはたいへんだろうね。／If it’s this hot, today’s outing will be terrible.
（4）こんなにたくさんの人ひとに見みられているんじゃ緊張きん ちょうしてしまうな。／I’ll be nervous if I’m being watched by this many people.
A spoken expression for (の)では.
