【おなじ】

1. ...とおなじ

［Nとおなじ］

［Vのとおなじ］

（1）このステレオはうちのと同おなじだ。／This stereo is the same as ours.
（2）この本ほんはあの本ほんと出版社しゅっ ぱん しゃが同おなじだ。／This book has the same publisher as that one.
（3）この点てんで妥協だ きょうすることはすべてをあきらめるのと同おなじことだ。／Compromising on this point is tantamount to abandoning everything.
（4）あの人ひとが食たべているのと同おなじものをください。／Please give me the same thing that person is eating.
（5）ヒンディー語ごは英語えい ごと同おなじインド・ヨーロッパ語族ご ぞくの言語げん ごだ。／Hindi is an Indo-European language, just like English.
Means that two objects or things are equivalent.

2. おなじV-る　なら／のだったら

（1）同おなじ買かうなら、少々しょう しょう高たかくても長持なが もちするものの方ほうがいい。／If I am going to buy one, I want something that lasts for a long time, even if it is a little more expensive.
（2）久ひさしぶりの旅行りょ こうなんだから、同おなじ行いくんだったら思おもい切きって遠とおくに行いきたいな。／This is my first trip in a while, so as long as I’m going, I want to take the plunge and travel somewhere really far away.
（3）同おなじお金かねをかけるのなら、食たべてなくなるものでなく、いつまでも使つかえるものにかける方ほうが意味い みがあると思おもう。／If I’m going to spend the money, I think it makes more sense to spend it on something I can use for a long time rather than something to eat, which doesn’t last.
（4）A:一緒いっ しょにフランス語ごか何なにか習ならいに行いかない？／Shall we take French classes or something together?
B:そうねえ、フランス語ごもいいけど、同おなじ習ならうんだったら人ひとのやってないような言語げん ごの方ほうがいいと思おもわない？／Well, sure, French would be fine, but if we’re going to study a language, don’t you think it would be better to study one that no one else does?
Means “as long as one is going to do a particular thing.” When there are many ways to do something, this pattern is used to state the best way. Similar to どうせなら and せっかくなら.
